The web application that will be implemented is the registration of a user in our cultural fest "SHADOW" and also selection of the events of his/ her choice and add them to a cart and generate a pass.<br>
          Firstly, the user will have to register themselves in the "Register" webpage by providing their personal information after which he/ she can select the events from the "Events" webpage and add them to a cart, where they will recieve the sub total. <br><br>The relations or tables used for storing the user input and the cost of the events will be as follows: <br><br>
        Personal information: (First Name, Last name, Gender, Email ID, Password, College Name, Phone Number)<br>
        Events Selected: (Email ID, Event ID)
All Events: (Event ID, Event Name, Event Cost, Event Catagory)
